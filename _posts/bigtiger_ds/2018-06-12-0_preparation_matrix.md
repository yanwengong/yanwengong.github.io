## Matrix Vector

$$A_{ij}$$ represents a matrix with i row and j column - n x j matrix

## Square Zero Identity

1. special matrix
  * square matrix: n rows and n columns - n x n matrix
  * zero matrix: all elements are 0
  * identity matrix, example:
$$\begin{array}
{rrr}
1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
\end{array}  
$$
